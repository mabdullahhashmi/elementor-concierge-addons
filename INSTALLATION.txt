📦 WHAT'S INCLUDED
──────────────────────────────────────────────────────────────────────────────

This plugin provides custom Elementor widgets for tour pages:

  ✓ Sidebar Nav Title    - A title header for your sidebar navigation
  ✓ Sidebar Nav          - Interactive section navigation with scroll-spy
  ✓ Ready for future widgets (plugin structure supports multiple addons)


🚀 INSTALLATION STEPS
──────────────────────────────────────────────────────────────────────────────

1. UPLOAD THE PLUGIN
   - Copy entire "elementor-concierge-addons" folder to:
     /wp-content/plugins/
   
   Final path should be:
     /wp-content/plugins/elementor-concierge-addons/
                          elementor-concierge-addons.php

2. ACTIVATE IN WORDPRESS
   - Go to WordPress Dashboard → Plugins
   - Find "Elementor Concierge Addons"
   - Click "Activate"

3. VERIFY IT WORKS
   - Edit any page with Elementor
   - Left sidebar → Look for "Concierge Golf" category
   - You should see 2 widgets:
     • Sidebar Nav Title
     • Sidebar Nav

4. YOU'RE READY!
   - Drag widgets to your page
   - Configure in the right panel
   - See README.md for detailed usage


📋 REQUIREMENTS

⚙️ BASIC CONFIGURATION
──────────────────────────────────────────────────────────────────────────────

SIDEBAR NAV TITLE
  • Add widget to page
  • Set title text (e.g., "On this page")
  • Choose background color (default: Navy #1d3461)
  • Customize font & padding

SIDEBAR NAV
  • Add widget to page
  • Add navigation items (Label + Section ID)
    Example: Label="Trip Highlights", Link="#highlights"
  • Enable "Scroll Spy" for auto-highlighting
  • Customize colors and spacing

IMPORTANT: Your page sections must have matching IDs:
  <section id="highlights">...</section>
  <section id="itinerary">...</section>
  etc.

🆘 COMMON ISSUES
──────────────────────────────────────────────────────────────────────────────

Q: Plugin doesn't appear in WordPress?
A: Make sure Elementor is installed and activated first.
   Then install this plugin.

Q: Widgets don't show in Elementor?
A: Clear Elementor cache: Elementor → Tools → Regenerate CSS

Q: Scroll spy not working?
A: Check section IDs match navigation links (case-sensitive)
   Add data-scroll-offset="150" (increase if header is tall)

Q: Colors not applying?
A: Clear browser cache
   Check that valid color codes are used (#hex or rgb)

For more help, see README.md or DEVELOPMENT.md
